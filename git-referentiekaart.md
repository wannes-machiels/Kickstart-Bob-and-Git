# Git — Snelle Referentiekaart

| Commando | Wat doet het? |
| --- | --- |
| `git init` | Nieuw Git-project starten |
| `git clone <url>` | Bestaand project downloaden |
| `git status` | Status van bestanden bekijken |
| `git add <bestand>` | Bestand stagen voor commit |
| `git add .` | Alle wijzigingen stagen |
| `git commit -m "..."` | Gestagede wijzigingen opslaan |
| `git log --oneline` | Compacte commitgeschiedenis |
| `git push` | Commits naar remote sturen |
| `git pull` | Wijzigingen van remote ophalen |
| `git fetch` | Wijzigingen ophalen (niet samenvoegen) |
| `git branch` | Branches bekijken |
| `git switch -c <naam>` | Nieuwe branch aanmaken en wisselen |
| `git checkout -b <naam>` | Nieuwe branch aanmaken en wisselen |
| `git merge <branch>` | Branch samenvoegen |
| `git diff` | Wijzigingen bekijken |
| `git stash` | Wijzigingen tijdelijk opzijzetten |
| `git stash pop` | Opzijgezette wijzigingen terughalen |
| `git reset --soft HEAD~1` | Laatste commit ongedaan maken |
